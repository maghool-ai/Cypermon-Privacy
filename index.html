سياسة الخصوصية – CyperMon

نحن نقدر خصوصيتك ونلتزم بحماية بياناتك. توضح هذه السياسة كيفية تعامل تطبيق CyperMon مع المعلومات أثناء استخدامك للتطبيق.

١. البيانات التي نقوم بجمعها

لا يقوم تطبيق CyperMon بجمع أي بيانات شخصية أو تتبع نشاط المستخدم داخل التطبيق، باستثناء المعلومات التي يقدمها المستخدم عند إنشاء حساب.

البيانات التي قد تُجمع:

  • البريد الإلكتروني – عند إنشاء حساب أو تسجيل الدخول.
  • اسم المستخدم داخل اللعبة.

لا نقوم بجمع أي معلومات أخرى مثل الموقع الجغرافي، الصور، جهات الاتصال، أو بيانات الجهاز.

٢. كيف نستخدم البيانات

يتم استخدام البريد الإلكتروني واسم المستخدم فقط من أجل:

  • تسجيل دخول اللاعب.
  • حفظ تقدم اللاعب داخل اللعبة.

لا يتم استخدام البيانات لأي أغراض إعلانية أو تتبع أو تحليل.

٣. مشاركة البيانات مع أطراف خارجية

لا نقوم ببيع أو مشاركة بيانات المستخدم مع أي شركة خارجية.

نستخدم خدمة Firebase Authentication لأغراض تسجيل الدخول فقط، وهي لا تستخدم البيانات لتتبع المستخدم عبر التطبيقات الأخرى.

٤. حماية المعلومات

نستخدم تقنيات آمنة لحماية بيانات المستخدم، ولا نقوم بتخزين أي بيانات حساسة على خوادم خارجية غير Firebase.

Privacy Policy – CyperMon

We value your privacy and are committed to protecting your personal data. This Privacy Policy explains how the CyperMon app handles information while you use it.

1. Information We Collect

CyperMon does not collect personal data or track users' activity, except for the information voluntarily provided when creating an account.

Data we may collect:

  • Email address – when creating an account or logging in.
  • In-game username.

We do NOT collect location data, photos, contacts, device information, or browsing data.

2. How We Use the Information

The email and username are used only for:

  • Authenticating and logging the user into the game.
  • Saving and restoring gameplay progress.

We do not use the data for advertising, analytics, or tracking.

3. Sharing Data with Third Parties

We do not sell or share user data with any external company.

We use Firebase Authentication solely for login purposes. Firebase Auth does not track users across apps or websites.

4. Data Security

We use secure methods to protect user information, and no sensitive data is stored outside Firebase services.
